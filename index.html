Gama-Soft Philosophy

Gamasoft mission is to help organisations to build better software products.People love to complain about their legacy systems, but if you have a legacy system, it means you have a working, successful system which deliver real value to business, and this is definitely a good thing.
Even more, legacy system is a system so successful that it makes very hard to dispose of it, even when it is obsolete in some (or all) respects.We believe is it possible to keep legacy applications nimble and easy to maintain and expand.
It could be not easy, but is cheaper and safer than trying rewrite from scratch..
